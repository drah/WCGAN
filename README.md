# UTBIT
## Prepare the data:
1. Download CelebA dataset
1. Place the data like the following:
    * celeba/
        * img_align_celeba/
             * all the images are put here
        * list_attr_celeba.txt
        * vocab.txt
    * main.py
    * model.py
    * module.py
    * utils.py

## Start to train:
* python main.py

## Start to test:
* python main.py --phase test
